## 📝 Content Notes

### Placeholder Text
Since exact body text from the reference images wasn't available, descriptive placeholder text has been used that matches the context and tone of each section. 

**To update with actual content:**
1. Open `index.html`
2. Find the section by ID (e.g., `id="about"`)
3. Replace paragraph text with your actual content
4. Save and refresh browser

### SVG Illustrations
All illustrations are custom-created placeholder SVGs with hand-drawn aesthetic matching the described style:
- Single-weight dark purple strokes (#2D1B44)
- Simplified but recognizable figures
- Consistent style across all sections

**To replace with actual images:**
1. Export your images as SVG or PNG
2. Place in `assets/images/` folder
3. Replace inline SVG code with `<img>` tags
4. Update alt text appropriately

## 🔧 Customization Guide

### Change Colors
Edit `assets/css/styles.css`:
```css
:root {
  --color-off-white: #FAF9F6;
  --color-deep-purple: #4B246C;
  --color-dark-purple: #2D1B44;
  --color-accent-yellow: #FFD700;
}
```

### Modify Typography
In `index.html`, change Google Fonts link and update Tailwind classes.

### Add New Sections
1. Copy existing section structure
2. Update section ID
3. Add to navigation menu
4. Create illustration
5. Update footer links
