## 📦 Deployment to GitHub Pages

### Prerequisites

1. GitHub account
2. Git installed on your computer

### Deployment Steps

#### Option 1: Using GitHub Desktop (Easier)

1. **Download GitHub Desktop**: https://desktop.github.com/
2. **Install and sign in** with your GitHub account
3. **Add existing repository**:
   - Click "File" → "Add Local Repository"
   - Browse to: `c:\xampp\htdocs\Infaratech VA`
   - Click "Add Repository"
4. **Publish to GitHub**:
   - Click "Publish repository" button
   - Enter repository name: `infratech-va`
   - Make it **Public**
   - Click "Publish Repository"
5. **Enable GitHub Pages**:
   - Go to: https://github.com/YOUR_USERNAME/infratech-va/settings/pages
   - Under "Source", select "Deploy from a branch"
   - Select branch: `main` or `master`
   - Select folder: `/ (root)`
   - Click "Save"
6. **Wait 2-5 minutes** for deployment
7. **Access your site** at: `https://YOUR_USERNAME.github.io/infratech-va/`

#### Option 2: Using Git Command Line

1. **Open PowerShell** in the project folder:
   ```powershell
   cd "c:\xampp\htdocs\Infaratech VA"
   ```

2. **Initialize Git repository**:
   ```powershell
   git init
   ```

3. **Add all files**:
   ```powershell
   git add .
   ```

4. **Commit changes**:
   ```powershell
   git commit -m "Initial commit - InfraTech VA website"
   ```

5. **Create repository on GitHub**:
   - Go to: https://github.com/new
   - Repository name: `infratech-va`
   - Make it **Public**
   - Click "Create repository"

6. **Connect local to remote**:
   ```powershell
   git remote add origin https://github.com/YOUR_USERNAME/infratech-va.git
   ```

7. **Push to GitHub**:
   ```powershell
   git branch -M main
   git push -u origin main
   ```

8. **Enable GitHub Pages**:
   - Go to repository Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main`, Folder: `/ (root)`
   - Click Save

9. **Wait 2-5 minutes** for deployment

10. **Your site is live** at: `https://YOUR_USERNAME.github.io/infratech-va/`
